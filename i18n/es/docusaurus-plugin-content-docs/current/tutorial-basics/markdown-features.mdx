# Características de Markdown

Docusaurus viene con **Markdown mejorado** que te permite escribir contenido rico.

## Frontmatter

Cada archivo Markdown puede tener un frontmatter YAML en la parte superior:

```yaml
---
id: my-doc
title: Mi Documento
description: Descripción de mi documento
slug: /my-custom-url
---

# Mi Documento

Contenido del documento...
```

## Markdown JSX

Puedes usar **JSX** en tus archivos Markdown para crear contenido interactivo:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '2px',
      color: '#fff',
      padding: '0.2rem',
    }}>
    {children}
  </span>
);

<Highlight color="#25c2a0">Docusaurus es genial!</Highlight>
```

## Tabs

Si quieres crear pestañas, puedes usar el componente `Tabs`:

```jsx
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="apple" label="Apple" default>
    Esta es una manzana 🍎
  </TabItem>
  <TabItem value="orange" label="Orange">
    Esta es una naranja 🍊
  </TabItem>
  <TabItem value="banana" label="Banana">
    Esta es una banana 🍌
  </TabItem>
</Tabs>
```

## Código

Puedes escribir código con resaltado de sintaxis:

```jsx
function hello() {
  console.log('Hello world!');
}
```
